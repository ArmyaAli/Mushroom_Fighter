# Mushroom_Fighter
A small game written with raylib

ROADMAP -> 
* IMPLEMENT A BASIC MENU (IN PROGRESS)
* GIVE PLAYER A HEALTH BAR AND STATE
* BATTLE MECHANICS VERSUS ANOTHER PLAYER
* BE ABLE TO CREATE MULTIPLE PLAYERS AND ALL FIGHT AT ONCE
* REFACTOR CODE INTO MODULAR HEADER FIELS AND SOURCES FILE (CLEANUP)
* IMPLEMENT UDP SOCKETS FOR REAL TIME POSITION UPDATES OF THE PLAYER ON DIFFERENT NETWORKS
* HANDLE COLLISIONS OVER NETWORK
* USE A BIGGER BETTER SPRITESHEET
